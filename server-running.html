Storiats - Local Server Running

Your Storiats application is now running locally. You can access it at:

Open Storiats (http://localhost:3000)

If port 3000 is in use, try these alternate ports:

  • http://localhost:3001
  • http://localhost:3002
  • http://localhost:3003

Troubleshooting

If you encounter any issues:

  • Check the terminal output for error messages
  • Make sure all required environment variables are set
  • Try restarting the development server with npm run dev
